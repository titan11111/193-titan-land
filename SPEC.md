## ゲート② 設計書

### コンセプト

小さな巨人「ティト」が、エナジージェムを集めながらビーコンタワーを目指す横スクロールアクション。FC様式準拠。

### 世界観（オリジナルIP）

- **主人公 ティト（TITO）**: 青いからだ＋シアンのリムライトの小さな巨人。タイタンズカラー（--fc-blue #0058f8 系）
- **敵**:
  - **ドテコ**: 転がり歩く岩の子。踏むとつぶれる（踏みつけ可）
  - **ハリゴロ**: トゲだらけの歩行球。踏むとダメージ（横からも上からも危険）
- **アイテム**:
  - **エナジージェム**（黄）: 1個100点。100個で1UP
  - **パワーコア**（?ブロックから出現）: シールド獲得（1回だけ被弾を無効化。見た目=リムライトが金色に）
- **ゴール**: ステージ最奥の**ビーコンタワー**に触れるとクリア（高い位置ほどボーナス）

### 画面・様式（retro-style-guide 10条準拠）

- 論理解像度 **256×240**、`image-rendering: pixelated`、**整数倍スケールのみ**（floor）
- 全描画は整数座標スナップ（`|0`）。**arc/beginPath禁止**、全スプライトはドット定義（文字列マップ→オフスクリーンcanvas）
- スプライト等倍描画のみ（拡大縮小なし）。1論理ドット=1canvasドット
- パレット: game-fc NESカラーを基準に、色相ごと明度3段ランプ。**ティトは黒輪郭＋シアンリムで背景から浮かせる**
- パララックス: 遠景（雲/星 0.25×）＋中景（丘/岩柱 0.5×）＋地面（1.0×、16pxタイル繰り返し）
- HUD: 自作8×8ビットマップフォント（数字＋英大文字）。上部固定・ゼロ埋め（SCORE 000000 / GEM 00 / TIME 300 / WORLD 1-1）
- タイトルは「TITAN LAND」ロゴをドットで表示、「PRESS START」8フレーム周期点滅
- スキャンライン: body::after オーバーレイ（game-fc標準）

### ステージ構成

| ステージ | テーマ | 長さ | 特徴 |
|---|---|---|---|
| 1-1 | 草原（昼・水色空） | 約220タイル | 基本文法の教示: 段差→ドテコ→?ブロック→穴→ハリゴロ |
| 1-2 | 洞窟（夜・紺空） | 約200タイル | 足場が細かく穴が増える。ジェム回廊のリズム配置 |
| クリア | THANK YOU画面 | - | スコア表示→タイトルへ |

### 物理パラメータ（60fps基準・px/frame）

- 重力 0.22（ジャンプボタン保持中 0.12 = 可変ジャンプ）
- 初速 -5.0 → 最大跳躍 約56px ≒ **3.5タイル**
- 走り: 加速 0.08 / 最高速 2.0 / 摩擦 0.90
- コヨーテタイム 6フレーム、ジャンプ先行入力 6フレーム
- 踏みつけ判定: 落下中＆敵の上1/3に接触 → 敵撃破＋小バウンド(-3.0)

### ルール

- 残機3。被弾（シールドなし）・落下・TIME切れで1ミス。0でGAME OVER
- TIME 300から1秒ごと減少。クリア時 残TIME×10点ボーナス
- スコア: ジェム100 / ドテコ踏み200 / パワーコア1000 / タワーボーナス最大2000

### サウンド（WebAudio・3声以上）

- BGM: 矩形波メロ＋三角波ベース＋ノイズドラムの16分シーケンサ（1-1=長調140bpm、1-2=短調・低め）
- ジングル: クリア／ミス／GAME OVER
- SE: ジャンプ（矩形波スイープ）・踏みつけ・ジェム・パワーコア・被弾・タワー到達
- iOS: AudioContextシングルトン＋pointerdown/onceでunlock

### 操作系（ios-game-base準拠）

- キーボード: ←→移動、Z/Space=ジャンプ、P=ポーズ
- タッチ: 左下に←→ボタン、右下にJUMPボタン（大）。bindPointer＋setPointerCapture
- viewport-fit=cover、safe-area-inset、ダブルタップ/長押し/ピンチ防止、overscroll-behavior:none
- 縦持ち想定（canvas上部・パッド下部）。横向きでも整数倍フィットで動作

### 技術・配信

- 単一 `index.html`（CSS/JSインライン、画像ファイルなし）。目標 <100KB ≪ 3MB鉄則
- 外部リソースはGoogle Fonts（Press Start 2P、HTML UI部のみ）1件
- 公開: GitHub Pages（/game-publish）→ Slack #general 通知
